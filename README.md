# DialReturn Landing Page

A fast, static single-page landing page for DialReturn, built with plain HTML, CSS, and JavaScript.

## Files

- `index.html` - Page content and structure
- `styles.css` - Responsive dark SaaS styling
- `script.js` - Mobile navigation, scroll animations, FAQ/legal link helpers, and form demo state

## GitHub Pages

Publish this folder directly from the repository root or from a `docs` folder. No build step, backend, or package install is required.

Before production launch, replace the placeholder Privacy Policy and Terms of Service text with your final legal language and connect the contact form to your preferred form service or CRM.
